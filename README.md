# Projet-Websockets
Le fichier projectElectron a ete code en electron et possede un serveur websockets dans lequel plusieur personne ce connecter et comminique comme dans une messagerie classique

Le fichier projectFlask a ete code en flask et a les meme caracteristique que le le fichier electron mais il a ete code en flask


Ces deux fichier s'inscrive dans un mini-projet multi polyvalent dans lequel il faut faire communique plusieur User(Utilisateur) dans une messagerie local en temps reel.
